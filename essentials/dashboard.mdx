## Overview

The Dashboard is your SmartAlex command center. It provides a quick overview of your account status, key metrics, and guides you through setup.

## Setup Checklist

New accounts see a setup checklist guiding you through the essential first steps:

<Steps>
  <Step title="Create Your First Agent">
    Build your AI voice agent with custom personality and knowledge.

    **Status indicators:**
    - Gray circle: Not started
    - Green check: Completed
  </Step>

  <Step title="Get a Phone Number">
    Provision a phone number for your agent to receive calls.
  </Step>

  <Step title="Test Your Agent">
    Use Quick Test to verify your agent works as expected.
  </Step>

  <Step title="Assign Number to Agent">
    Connect your phone number to your agent to go live.
  </Step>
</Steps>

Once all steps are complete, the checklist is replaced with your analytics dashboard.

## Key Metrics

After setup, your dashboard displays:

### Call Summary

| Metric | Description |
|--------|-------------|
| **Total Calls** | All calls in the selected period |
| **Inbound Calls** | Calls received on your numbers |
| **Outbound Calls** | Campaign calls made |
| **Web Calls** | Calls from website widgets |

### Performance Metrics

| Metric | Description |
|--------|-------------|
| **Average Duration** | Mean call length |
| **Connect Rate** | Percentage of answered calls |
| **Sentiment Score** | Positive vs. negative call ratio |

### Cost Overview

| Metric | Description |
|--------|-------------|
| **Current Usage** | Minutes used this billing period |
| **Estimated Cost** | Projected monthly cost |
| **Wallet Balance** | Prepaid credit remaining |

## Time Period Selection

View metrics for different time periods:
- **Today** - Current day
- **Yesterday** - Previous day
- **Last 7 days** - Weekly view
- **Last 30 days** - Monthly view
- **Custom range** - Select specific dates

## Quick Actions

The dashboard provides shortcuts to common tasks:

| Action | Description |
|--------|-------------|
| **Create Agent** | Start the agent creation wizard |
| **Get Number** | Provision a new phone number |
| **Start Campaign** | Create a new outbound campaign |
| **View Calls** | Jump to call logs |

## Recent Activity

See your most recent calls at a glance:
- Caller/recipient name or number
- Call direction (inbound/outbound/web)
- Duration and outcome
- Sentiment indicator
- Click to view full details

## Notifications

System notifications appear on the dashboard:
- Campaign completions
- Usage alerts
- Billing reminders
- New feature announcements
