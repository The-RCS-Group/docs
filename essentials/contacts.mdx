## Importing Contacts

Upload contacts via CSV with the following fields:

| Field | Required | Description |
|-------|----------|-------------|
| `phone` | Yes | Phone number |
| `first_name` | No | First name |
| `last_name` | No | Last name |
| `email` | No | Email address |
| `company` | No | Company name |

## Contact Management

### Viewing Contacts

Navigate to **Contacts** to see all your contacts. You can:
- Search by name or phone
- Filter by status (active, called, failed)
- View call history per contact
- Delete contacts

### Contact Status

| Status | Description |
|--------|-------------|
| **Pending** | Not yet called |
| **Calling** | Currently on a call |
| **Completed** | Call finished successfully |
| **Failed** | Call could not be completed |
| **Callback** | Scheduled for retry |

## DNC (Do Not Call) List

SmartAlex respects Do Not Call lists. Contacts can be flagged to prevent future calls.
