## Overview

SmartAlex Analytics gives you deep insights into your AI agent's performance. Track call volumes, sentiment trends, costs, and more.

## Accessing Analytics

Navigate to **Analytics** in your dashboard to view:
- Call volume trends
- Sentiment analysis
- Campaign performance
- Cost tracking
- Period comparisons

## Call Volume

### Metrics

| Metric | Description |
|--------|-------------|
| **Total Calls** | All call attempts |
| **Completed Calls** | Successfully connected calls |
| **Missed Calls** | No-answer, busy, failed |
| **Voicemails** | Calls that went to voicemail |

### Trends

View call volume over time:
- **Daily view** - Hour-by-hour breakdown
- **Weekly view** - Day-by-day comparison
- **Monthly view** - Daily trends over 30 days

Identify peak calling times to optimize agent availability and campaign scheduling.

## Sentiment Analysis

Track the emotional tone of your calls:

### Sentiment Breakdown

| Sentiment | Meaning |
|-----------|---------|
| **Positive** | Customer expressed satisfaction, interest, or gratitude |
| **Neutral** | Standard information exchange, no strong emotion |
| **Negative** | Frustration, complaints, or dissatisfaction |

### Sentiment Trends

Monitor sentiment over time to:
- Identify improvements in customer satisfaction
- Spot issues before they become problems
- Measure the impact of agent prompt changes
- Track seasonal variations

### Sentiment by Agent

Compare sentiment across agents to identify:
- Best-performing agent configurations
- Agents that need prompt improvements
- Industry-specific patterns

## Campaign Analytics

For each campaign, view:

| Metric | Description |
|--------|-------------|
| **Contacts Reached** | Successfully connected calls |
| **Connect Rate** | Percentage of answered calls |
| **Average Duration** | Mean call length |
| **Sentiment Distribution** | Positive/neutral/negative breakdown |
| **Total Cost** | Campaign calling expenses |

### Campaign Comparison

Compare multiple campaigns side-by-side:
- Performance metrics
- Cost efficiency
- Best performing times
- Contact list quality indicators

## Cost Tracking

### Usage Metrics

| Metric | Description |
|--------|-------------|
| **Minutes Used** | Total calling minutes |
| **Per-Minute Cost** | Average cost per minute |
| **Total Spend** | Current period expenses |
| **Projected Cost** | Estimated monthly total |

### Cost Breakdown

View costs by:
- **Call type** - Inbound, outbound, web
- **Agent** - Which agents are most expensive
- **Campaign** - Campaign-level costs
- **Day/Week** - Spending trends over time

### Budget Alerts

Set up alerts when:
- Daily spend exceeds threshold
- Weekly spend exceeds threshold
- Wallet balance is low
- Approaching plan limits

## Period Comparisons

Compare performance across time periods:

| Comparison | Use Case |
|------------|----------|
| **Week over Week** | Track weekly improvement |
| **Month over Month** | Monthly trends |
| **Custom Periods** | Campaign A vs Campaign B |

See percentage changes and trend indicators for key metrics.

## Insights

AI-generated insights highlight:

- **Opportunities** - Times when more capacity could help
- **Anomalies** - Unusual patterns to investigate
- **Recommendations** - Suggested prompt improvements
- **Trends** - Notable patterns in your data

## Exporting Reports

Export analytics data:

1. Select your date range
2. Choose metrics to include
3. Click **Export**
4. Download CSV or PDF report

Use exports for:
- Executive reporting
- External analysis
- Compliance documentation
- Billing verification
