## Authentication

All API endpoints require authentication using a Bearer token.

```bash
curl -X GET "https://api.smartalex.ai/v1/agents" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Get your API key from the **Settings > API** page in your SmartAlex dashboard.
