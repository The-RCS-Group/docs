# SmartAlex Documentation

Official documentation for SmartAlex - AI Voice Assistant Platform.

**Live docs:** [docs.getsmartalex.com](https://docs.getsmartalex.com)

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview documentation changes locally:

```bash
npm i -g mint
```

Run at the root of your documentation (where `docs.json` is located):

```bash
mint dev
```

View your local preview at `http://localhost:3000`.

## Publishing

Changes are deployed automatically after pushing to the main branch via the Mintlify GitHub app.
